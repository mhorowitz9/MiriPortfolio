Koalaship Project

Project Overview

Koalaship is a Figma prototype designed to simplify the scholarship search process, ensuring equitable opportunities for all students to pursue educational goals by alleviating financial constraints.

Project Details

  • Objective: Create a user-friendly interface for students seeking scholarships.
  • Role: Lead UX/UI Designer
  • Tools Used: Mural, Figma
  • Process:
    • Conducted user research including interviews and surveys with students.
    • Created personas and journey maps to understand user needs and pain points.
    • Developed wireframes and interactive prototypes in Figma.
    • Iteratively tested designs for usability and effectiveness.
  • Outcome: The Koalaship prototype features intuitive navigation, personalized scholarship recommendations, and a streamlined application process, enhancing accessibility to educational funding.
